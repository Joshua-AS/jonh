PROGRAM NewtonsLawsOfMotion

IMPLICIT NONE


INTEGER :: choice
REAL :: force, mass, acceleration, distance, time, velocity

! Prompt user to choose which Newton's law to execute

PRINT *, 'Which Newton law would you like to execute?'

PRINT *, '1. Newton''s First Law (Law of Inertia)'

PRINT *, '2. Newton''s Second Law (F = ma)'

PRINT *, '3. Newton''s Third Law (Action and Reaction)'

READ *, choice


! Execute the chosen Newton's law

SELECT CASE (choice)



CASE (1) ! Newton's First Law (Law of Inertia)

PRINT *, 'Newton''s First Law: An object will remain at rest or

in uniform motion unless acted upon by an external force.'



CASE (2) ! Newton's Second Law (F = ma)

PRINT *, 'Newton''s Second Law: Enter the mass (kg):'

READ *, mass



PRINT *, 'Enter the force (N):'

READ *, force



acceleration = force / mass

PRINT *, 'Acceleration:', acceleration, 'm/s^2'



CASE (3) ! Newton's Third Law (Action and Reaction)

PRINT *, 'Newton''s Third Law: For every action,

there is an equal and opposite reaction.'

! This law doesn't require any input values.



CASE DEFAULT

PRINT *, 'Invalid choice.'



END SELECT



END PROGRAM NewtonsLawsOfMotion

1.Program Structure: The program starts with the PROGRAM statement and ends with the END PROGRAM statement. It is named NewtonsLawsOfMotion.

2.Variable Declaration: The IMPLICIT NONE statement is used to ensure all variables are explicitly declared.

3.User Prompt: The user is prompted to choose which Newton's law to execute by entering a number corresponding to the desired law.

4.Selection Structure: A SELECT CASE structure is used to execute the chosen Newton's law based on the user's input.

5.Case Statements: Each case statement corresponds to one of Newton's laws. In each case, the program performs the necessary calculations or displays information related to the chosen law.

6.Input: For Newton's second law, the program prompts the user to input the mass and force values.

7.Calculation: The acceleration is calculated using the formula a=Fm,(Newton's second law).

8.Output: The calculated acceleration is printed to the console.

9.Default Case: An invalid choice by the user results in a default message indicating an invalid choice.
